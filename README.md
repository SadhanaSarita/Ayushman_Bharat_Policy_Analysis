AYUSHMAN BHARAT POLICY ANALYSIS

What This Project Is About?
This project analyses the efficiency of Ayushman Bharat – Pradhan Mantri Jan Arogya Yojana (AB-PMJAY), India's flagship universal health coverage scheme launched on 23 September 2018.

Research Question:
How efficiently has PMJAY achieved its stated goals of reducing out-of-pocket expenditure, expanding healthcare access, and ensuring cashless treatment for India's poorest families?

All data is sourced from primary official documents only. No secondary sources or estimates are used without explicit labelling.
